# @skatehubba/ui

Shared React Native UI components and theme for SkateHubba mobile app.

**Note**: This package is React Native-specific and should only be used by the mobile app. Web apps should create their own UI components.

## Exports

- `SKATE` - Theme constants (colors, radius, timing)
- `GrittyButton` - Reusable button component

## Usage

```typescript
import { SKATE, GrittyButton } from '@skatehubba/ui';
```
